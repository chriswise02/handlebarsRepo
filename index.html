Conditional Helpers

#if

Conditionally render a block. If its argument returns false, undefined, null, "", 0, or [], Handlebars will not render the block.

{{#if condition}}
  This will be rendered if condition is true.
{{/if}}
Conditional

#unless

The inverse of the if helper. Its block will be rendered if the expression returns a falsy value.

{{#unless condition}}
  This will be rendered if condition is false.
{{/unless}}

Iteration Helpers

#each

Iterate over a list using the built-in each helper. Inside the block, you can use this to reference the element being iterated over.

{{#each array}}
  {{this}}
{{/each}}
